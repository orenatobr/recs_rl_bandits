# Recs, Bandits & RL

This project implements:
- Content-based filtering
- Collaborative filtering
- Hybrid recommenders

## Getting Started
```bash
poetry install --no-root
```

## Example Usage
```bash
poetry run python main.py
```

## Folder Structure
- `recommender/`: recommendation algorithms
- `utils/`: metrics and tools
- `main.py`: entry point

## Requirements
- Python 3.11.8+
- numpy
- scikit-learn
